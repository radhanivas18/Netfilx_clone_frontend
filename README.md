# Netflix Clone Frontend

A responsive static Netflix landing-page clone built using HTML and CSS.

## 🔍 Project Overview
This page replicates the Netflix India landing experience with:
- Hero section with CTA and email input
- Trending posters grid
- Feature highlight cards
- FAQ-style accordion placeholders
- Footer with links and language selector

## 📁 Files
- `index.html` - main page structure
- `style.css` - styling and layout
- `assests/images/` - logos, background, posters

## 🚀 Run Locally
1. Open `index.html` in your browser.
2. Alternatively, run a local HTTP server (recommended):
   - Python 3: `python -m http.server 8000`
   - Then visit `http://localhost:8000`

## ✨ Notes
- This is static frontend markup and CSS only.
- You can customize posters and content from the HTML.
